Шаблоны для импорта востребованности профессий.

Формат «матрица» (рекомендуется): строки — профессии, столбцы — регионы, на перекрёстке 1 или 0.
- demand_matrix_import_template_wide.csv  — первая строка: название колонки профессий + регионы, далее строки: профессия, 0/1 по регионам.
- demand_matrix_import_template_wide.xlsx — то же в Excel (при импорте допускаются 0.0/1.0 из ячеек).

Построчный формат:
- demand_matrix_import_template_row.csv   — колонки: код_профессии, профессия, регион, востребована, год.

Востребована (значение «да»): 1, 1.0, да, true, yes, x, +. Нет: 0, 0.0, нет, false.
Разделитель CSV: ; или , (определяется автоматически). Кодировка: UTF-8 с BOM.

Сгенерировать шаблоны заново: python manage.py generate_demand_import_templates [--output-dir путь]
